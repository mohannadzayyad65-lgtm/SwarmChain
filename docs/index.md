## **نظرة عامة**

SwarmChain هو بلوكشين لامركزي يستخدم **الفيرومون** كآلية توافق ناشئة.

### **المميزات**
- حل سريع للشوك
- مقاومة السبام
- توازن حمل ذكي
- حوكمة لامركزية

### **التقنيات**
- Python + Rust
- Hybrid Consensus (PoS + ACO)
- On-Chain Governance
